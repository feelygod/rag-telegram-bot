# RAG Telegram Bot

Бот отвечает на вопросы на основе загруженных документов (RAG — Retrieval-Augmented Generation).

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

Создай файл `.env` или задай переменные окружения:

```env
TG_BOT_TOKEN=токен_от_@BotFather
OPENAI_API_KEY=твой_openai_ключ
```

> Если не указывать `OPENAI_API_KEY`, бот будет работать в режиме без LLM — просто показывать найденные чанки.

## Запуск

```bash
python bot.py
```

## Команды

- `/start` — приветствие
- `/stats` — статистика по базе
- `/clear` — очистить все документы

## Использование

1. Отправь боту `.txt` или `.md` файл — он будет разбит на чанки и проиндексирован
2. Задай вопрос текстом — бот найдёт релевантные чанки и ответит через LLM
